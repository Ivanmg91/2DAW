Conceptos CSS Complementarios

1. Transform - Transformaciones 2D

La propiedad transform permite mover, rotar, escalar o inclinar elementos sin afectar el flujo del documento.

Sintaxis básica:

transform: translate(x, y) | 
           translateX(x) | 
           translateY(y) |
            scale() | 
            rotate();

Translate - Mover elementos:

.elemento {
    transform: translate(50px, 100px); 
    /* Mueve 50px derecha, 100px abajo */
}

.centrado {
    transform: translateX(-50%); 
    /* Mueve hacia la izquierda el 50% de su ancho */
}

Ejemplo práctico - Centrado perfecto:

.contenedor {
    position: relative;
}

.caja-centrada {
    position: absolute;
    top: 50%;           /* Posiciona en el centro vertical */
    left: 50%;          /* Posiciona en el centro horizontal */
    transform: translate(-50%, -50%); 
    /* Ajusta por el tamaño del elemento */
}

💡 Clave: translate() usa el tamaño del propio elemento como referencia. -50% significa “muévete hacia atrás la mitad de tu tamaño”.

📝 Nota: Útil para Ejercicio 4 (centrado perfecto)

2. Transition - Transiciones suaves

Las transiciones crean animaciones suaves entre estados CSS (hover, focus, etc.).

Sintaxis básica:

transition: propiedad duración función-tiempo retraso;

Ejemplos básicos:

.boton {
    background: blue;
    transition: background 0.3s ease; /* Transición suave del color */
}

.boton:hover {
    background: red; /* El cambio será gradual */
}

Múltiples propiedades:

.elemento {
    opacity: 0;
    transform: translateY(20px);
    transition: opacity 0.3s ease, transform 0.3s ease;
}

.elemento:hover {
    opacity: 1;
    transform: translateY(0);
}

Funciones de tiempo comunes:

  • ease - Lento al inicio y final, rápido en medio
  • linear - Velocidad constante
  • ease-in - Lento al inicio
  • ease-out - Lento al final

💡 Clave: Las transiciones solo funcionan con propiedades que tienen valores numéricos (color, tamaño, posición, opacidad).

📝 Nota: Útil para Ejercicio 9 (efectos de aparición suave)

3. Media Queries - Diseño responsive

Las media queries permiten aplicar CSS solo cuando se cumplen ciertas condiciones (tamaño de pantalla, orientación, etc.).

Sintaxis básica:

@media (condición) {
    /* CSS que solo se aplica cuando se cumple la condición */
}

Ejemplos por tamaño de pantalla:

/* Estilos para pantallas pequeñas (móviles) */
@media (max-width: 768px) {
    .sidebar {
        width: 100%;
        float: none;
    }
}

/* Estilos para pantallas medianas (tablets) */
@media (min-width: 769px) and (max-width: 1024px) {
    .contenido {
        font-size: 16px;
    }
}

/* Estilos para pantallas grandes (desktop) */
@media (min-width: 1025px) {
    .contenedor {
        max-width: 1200px;
    }
}

Ejemplo práctico - Layout responsive:

/* Estilos por defecto (desktop) */
.sidebar {
    float: left;
    width: 200px;
}

.contenido {
    margin-left: 200px;
}

/* Adaptación para móviles */
@media (max-width: 768px) {
    .sidebar {
        float: none;      /* Quita la flotación */
        width: 100%;      /* Ocupa todo el ancho */
    }
    
    .contenido {
        margin-left: 0;   /* Quita el margen */
    }
}

Condiciones comunes:

  • max-width: 768px - Pantallas de hasta 768px (móviles)
  • min-width: 769px - Pantallas de 769px en adelante
  • orientation: portrait - Orientación vertical
  • orientation: landscape - Orientación horizontal

💡 Clave: Piensa “mobile first” - diseña primero para móvil y luego adapta para pantallas más grandes.

📝 Nota: Útil para Ejercicios 7, 8 y 10 (layouts que se adaptan a diferentes pantallas)

4. Combinando conceptos

Ejemplo: Menú responsive con animaciones

.menu {
    position: fixed;
    top: 0;
    left: -250px;                    /* Oculto por defecto */
    width: 250px;
    height: 100vh;
    background: #333;
    transform: translateX(0);        /* Posición inicial */
    transition: transform 0.3s ease; /* Transición suave */
}

.menu.abierto {
    transform: translateX(250px);    /* Se desliza hacia la derecha */
}

/* En móviles, el menú es diferente */
@media (max-width: 768px) {
    .menu {
        left: -100%;
        width: 100%;
    }
    
    .menu.abierto {
        transform: translateX(100%);
    }
}

Ejemplo: Tooltip responsive

.tooltip {
    position: absolute;
    bottom: 100%;
    left: 50%;
    transform: translateX(-50%);     /* Centrado horizontal */
    opacity: 0;                      /* Invisible por defecto */
    transition: opacity 0.2s ease;  /* Aparición suave */
}

.elemento:hover .tooltip {
    opacity: 1;                      /* Visible al hacer hover */
}

/* En móviles, tooltip más grande */
@media (max-width: 480px) {
    .tooltip {
        font-size: 14px;
        padding: 8px 12px;
        white-space: normal;         /* Permite salto de línea */
        max-width: 200px;
    }
}

💡 Tips para usar estos conceptos:

  1. Transform es tu amigo para centrado - Más flexible que márgenes negativos
  2. Transition hace todo más profesional - Añade suavidad a los cambios
  3. Media queries son esenciales hoy - Todo debe funcionar en móvil
  4. Combina conceptos gradualmente - No uses todo a la vez al principio
  5. Prueba en diferentes tamaños - Usa las herramientas de desarrollador del navegador
