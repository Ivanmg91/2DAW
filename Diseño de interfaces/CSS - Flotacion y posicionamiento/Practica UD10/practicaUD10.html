Ejercicios CSS: Flotación y Posicionamiento

Ejercicio 1: Contenedor Colapsado

Dificultad: ⭐⭐

Tienes este HTML y CSS. El contenedor .gallery no muestra su borde verde. ¿Por qué ocurre y cómo lo solucionas?

<div class="gallery">
  <img src="foto1.jpg" alt="Foto 1">
  <img src="foto2.jpg" alt="Foto 2">
  <img src="foto3.jpg" alt="Foto 3">
</div>

.gallery {
  border: 3px solid green;
  padding: 10px;
}

.gallery img {
  float: left;
  width: 30%;
  margin: 1%;
}

Tareas:

  1. Explica por qué el borde no se ve
  2. Proporciona 2 soluciones diferentes
  3. ¿Cuál es la solución moderna recomendada?

Ejercicio 3: El Misterio del Contenedor

Dificultad: ⭐⭐⭐

<div class="contenedor">
  <div class="caja-roja"></div>
  <p>Contenido del contenedor...</p>
</div>

.contenedor {
  width: 400px;
  height: 300px;
  background: lightblue;
  margin: 50px auto;
}

.caja-roja {
  position: absolute;
  top: 0;
  right: 0;
  width: 50px;
  height: 50px;
  background: red;
}

Tareas:

  1. Identifica el problema
  2. Corrígelo con el mínimo cambio posible
  3. Explica el concepto de “contenedor posicionado”

Ejercicio 6: Z-Index Caótico

Dificultad: ⭐⭐⭐⭐

Tienes 3 cajas superpuestas, pero el orden de apilamiento está mal. La caja verde debería estar encima de todo, luego la azul, y la roja al fondo.

<div class="escena">
  <div class="caja roja">Roja</div>
  <div class="caja azul">Azul</div>
  <div class="caja verde">Verde</div>
</div>

.escena {
  position: relative;
  width: 300px;
  height: 200px;
}

.caja {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  font-weight: bold;
  padding: 10px;
}

.roja { 
   background: red; 
   top: 20px; 
   left: 20px; 
   z-index: 100; 
   }
.azul { 
   background: blue; 
   top: 40px; 
   left: 40px; 
   z-index: 50; 
   }
.verde { 
   background: green; 
   top: 60px; 
	 left: 60px; 
   z-index: 10; 
  }

Pregunta trampa: ¿Por qué no basta con cambiar solo los valores de z-index?

Requisitos:

  • Usa shape-outside para el texto circular
  • Usa posicionamiento para el título superpuesto
  • El layout debe ser responsive

Ejercicio 8: Debugging Challenge

Dificultad: ⭐⭐⭐⭐⭐

Este código tiene múltiples problemas. Encuentra y corrige todos:

<div class="wrapper">
  <nav class="sidebar">
    <ul>
      <li><a href="#">Inicio</a></li>
      <li><a href="#">Productos</a></li>
    </ul>
  </nav>
  <main class="content">
    <div class="hero">
      <h1>Título Principal</h1>
      <button class="cta">
      ¡Comprar Ahora!</button>
    </div>
    <p>Contenido principal...</p>
  </main>
</div>

.wrapper {
  width: 100%;
}

.sidebar {
  float: left;
  width: 200px;
  background: #333;
  height: 100vh;
}

.content {
  margin-left: 200px;
}

.hero {
  position: relative;
  height: 400px;
  background: url('hero.jpg');
}

.cta {
  position: absolute;
  bottom: 20px;
  right: 20px;
  z-index: 999999;
}

Problemas a encontrar:

  1. El wrapper colapsa
  2. El botón CTA no se posiciona correctamente
  3. Uso inadecuado de z-index
  4. Falta de clearfix o alternativa moderna
  5. Problemas de responsive

Ejercicio 9: Tooltip Inteligente

Dificultad: ⭐⭐⭐⭐
